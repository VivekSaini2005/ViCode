═══════════════════════════════════════════════════════════════════════════
                         DO's and DON'Ts
═══════════════════════════════════════════════════════════════════════════

✅ DO: Press Enter key to create new lines
✅ DO: Type normally like in a text editor
✅ DO: Use the placeholder as a guide
✅ DO: Check the preview - you should see multiple lines

❌ DON'T: Type \n literally (like: 4\n1 2 3 4)
❌ DON'T: Use special characters for newlines
❌ DON'T: Copy-paste weird formats with escape sequences
